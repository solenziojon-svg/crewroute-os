# CrewRoute OS

**The complete operating system for serious landscaping operators.**

Built and battle-tested at CJS Landscape Solutions.

## What is CrewRoute OS?

CrewRoute OS is a fully automated landscaping operations system that replaces spreadsheets, manual dispatching, and guesswork with a clean, intelligent workflow.

### Core Features
- **Waste Score Engine** — Real-time scoring of every property (0-100) based on route overlap, crew idle time, fuel burn, and unbilled services.
- **Automated Morning Dispatch** — Runs at 6:30 AM every day. Highest penalty jobs first.
- **Tribe Score System** — Gamifies crew performance and builds real culture.
- **Ghost Page Audits** — One-click $497 fix plans that convert into long-term contracts.
- **Knight Dashboard** — Dark, professional admin view for the operator.

## Tech Stack
- Airtable (core database)
- Softr (front-end pages)
- Make.com (automation)
- OpenAI (5 core prompts)
- Telegram / ClickSend (crew communication)

## Quick Start

1. Duplicate the Airtable base (link in repo)
2. Connect to Softr
3. Paste the 5 AI prompts
4. Set up Make.com scenarios
5. Go live

## Repository Structure
- `/prompts` — All 5 core AI system prompts
- `/schema` — Airtable field maps
- `/components` — Ready-to-paste React components for Softr
- `/automation` — Make.com scenario blueprints
